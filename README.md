
# astrbot_plugin_galgame_box
## 简介
[AstrBot](https://github.com/AstrBotDevs/AstrBot)的社区插件，结合了VNDB和TouchGal的API，不但能够提供全面、细节的Galgame帮助信息，还能提供TouchGal的随机作品和资源下载等功能。后续会结合更多Gal网站的API提供更有趣的功能。


## 使用指南

本插件以 **`旮旯`** 作为主指令，后接空格与子指令即可调用各项功能。

##  功能列表

###  VNDB 相关
| 功能 | 指令格式 |别名| 说明 |
|------|----------|------|------|
| 指令查询 | `/旮旯` |`gal`、`GAL`| 指令组，查询子指令信息 |
| 作品查询 | `/旮旯 作品 <作品名>` | `游戏`|搜索指定作品 |
| 角色查询 | `/旮旯 角色 <角色名>` | `人物`|搜索指定角色 |
| 厂商查询 | `/旮旯 厂商 <厂商名>` | `作者`|搜索指定厂商 |
| ID 查询 | `/旮旯 ID <VNDB ID>` | `ID`|通过 VNDB ID 直接查询 |

###  TouchGal 相关
> 因为网站异常，此版本未测试此类功能是否正常工作。

| 功能 | 指令格式 |别名| 说明 |
|------|----------|---|---|
| 随机作品 | `/旮旯 随机` | |随机获取一部作品 |
| 资源下载 | `/旮旯 下载 <内容>` | `资源`|获取游戏资源 |

**关于下载参数 `<内容>`：**
- 可以是 TouchGal 纯数字 ID
- 可以是 VNDB ID
- 其他情况下，将通过 TouchGal 搜索该内容并让用户选择结果

###  AnimeTrace 相关
| 功能 | 指令格式 |别名|  说明 |
|------|----------|--|----|
| 角色出处识别 | `/旮旯 出处 <图片或图片链接>` |识别| 识别角色出处

**使用说明：**
- 如不填写参数，将开启会话等待用户发送图片
- 会话中不再接受图片链接

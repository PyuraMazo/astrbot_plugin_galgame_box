
# Galgame百宝盒
## 简介
[AstrBot](https://github.com/AstrBotDevs/AstrBot)的社区插件，本插件几乎满足了用户的所有与Gal相关的需求。本插件结合了VNDB、TouchGal、AnimeTrace、Steam的API，不但能够提供**全面且细致**的Galgame基础查询信息，而且能提供**随机作品推荐**、**资源下载**、**角色识别**、**Gal游戏拼图**等额外功能。

## 使用指南

本插件以 **`旮旯`** 作为主指令，后接空格与子指令即可调用各项功能。

> 本插件几乎所有结果反馈都是以图片形式呈现，因此使用了AstrBot官方的渲染服务，此渲染服务默认渲染服务器的响应时间较长。如需缩短渲染时间以优化使用体验，请根据[官方部署教程](https://docs.astrbot.app/others/self-host-t2i.html)部署本地渲染服务。

##  功能列表

> 演示使用 **`/`** 作为唤醒符

###  VNDB 相关
| 功能 | 指令格式 |别名| 说明 |
|------|----------|------|------|
| 指令查询 | `/旮旯` |`gal`、`GAL`| 指令组，查询子指令信息 |
| 作品查询 | `/旮旯 作品 <作品名>` | `游戏`|搜索指定作品 |
| 角色查询 | `/旮旯 角色 <角色名>` | `人物`|搜索指定角色 |
| 厂商查询 | `/旮旯 厂商 <厂商名>` | `作者`|搜索指定厂商 |
| ID 查询 | `/旮旯 ID <VNDB ID>` | `id`|通过 VNDB ID 直接查询 |

**关于ID查询参数 `<VNDB ID>`：**
- 目前插件仅支持通过VNDB ID查询作品、角色和厂商

###  TouchGal 相关

| 功能 | 指令格式 |别名| 说明 |
|------|----------|---|---|
| 随机作品 | `/旮旯 随机` | |随机获取一部作品 |
| 资源下载 | `/旮旯 下载 <内容>` | `资源`|获取游戏资源 |
| 推荐作品 | `/旮旯 推荐 <标签>` | `标签`|通过标签获取作品 |

**关于资源下载参数 `<内容>`：**
- 可以是 TouchGal 纯数字 ID
- 可以是 VNDB ID
- 其他情况下，将通过 TouchGal 搜索该内容并让用户选择结果

**关于推荐作品参数 `<标签>`：**
- 必填，否则请使用**随机作品**指令
- 标签可以是多个，多个标签请用**空格**分隔

###  AnimeTrace 相关
| 功能 | 指令格式 |别名|  说明 |
|------|----------|--|----|
| 角色识别 | `/旮旯 出处 <图片链接>` |`识别`| 智能图像识别

**使用说明：**
- 如不填写参数，将开启会话等待用户发送图片
- **图片链接**只能在调用指令时参数传入，会话中不接受图片链接
- 支持引用一张图片进行识别

###  Steam 相关
| 功能 | 指令格式 |别名|  说明 |
|------|----------|--|----|
| 绑定Steam | `/旮旯 绑定` || 只能响应私聊频道
| 展示拼图 | `/旮旯 拼图` |`steam`| 输出基于成就达成率的拼图
